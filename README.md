# 🚗 Scratch Car Race

Top-down autós játék Scratch-ben, **smooth (tehetetlenséges)** irányítással, minimappel.

---

## 🎮 Irányítás

- **W** – gáz
- **S** – fék
- **A** – balra fordulás
- **D** – jobbra fordulás
- **H** – duda

---

## ✨ Funkciók

- Smooth gyorsítás / lassulás (súrlódás)
- Hangeffektek: duda
- Minimap: a kocsi pozíciója piros ponttal követhető
